# Mad Birds

A basic **Unity** 2D game inspired by Angry Birds. The player launches birds to destroy structures and defeat enemies.

## Overview

Mad Birds is a simple physics-based slingshot game built in Unity as a personal learning project. It recreates the core mechanic of the classic Angry Birds game — aiming and launching projectiles to knock down obstacles and targets.

## Features

- Physics-based slingshot launching mechanic
- - 2D gameplay with Unity physics
  - - Scene-based level structure
    - - Built with Unity
     
      - ## Technologies
     
      - - Unity (2D)
        - - C#
